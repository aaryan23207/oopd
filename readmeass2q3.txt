# Library Borrowing System

This is a C++ program that allows users to borrow books, magazines, and journals from a library. It provides a user-friendly interface for searching and borrowing library materials.

## Usage

1. Run the program by executing the compiled binary.

2. Choose whether you are a registered user (1 for yes, 2 for no) and whether you are a student or faculty member (1 for student, 2 for faculty).

3. If you are a registered user, enter your password (default password is "1234").

4. Select the type of material you want to borrow (1 for books, 2 for magazines, 3 for journals).

5. Follow the on-screen prompts to search for and borrow the desired material.

## Dependencies

- This program uses C++ standard libraries and does not require external dependencies.

## Code Structure

- The program is organized into classes: `Books`, `M`, and `J`, each responsible for handling a specific type of material.
