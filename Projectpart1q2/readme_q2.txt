# C++ Telecom Data Analyzer

This C++ program is designed to analyze subscriber data from a CSV file. It uses the "Projectq1-5.h" header file to define classes and structures for data analysis. Below, you'll find information on how to use this program.

## Prerequisites

To run this program, you need to have a C++ compiler installed on your system. The program uses standard C++ libraries, so no additional dependencies are required.

## Getting Started

1. Clone this repository to your local machine.

2. Compile the program using your preferred C++ compiler:

    ```shell
    g++ main.cpp -o telecom_data_analyzer
    ```

3. Run the program:

    ```shell
    ./telecom_data_analyzer
    ```

## Program Features

### Subscriber Data Analysis

- The program reads subscriber data from a CSV file ("subscribers.csv").
- You can choose to print all subscriber data or search for specific information based on a date, company, location, or the number of subscribers.

## Usage

Follow the on-screen instructions to use the program's features. The program will guide you through data input and provide search options for subscriber data.

## Source Files

- The program consists of a header file "Projectq1-5.h" that defines the structures and classes used in the program.
- The main program logic is in "main.cpp."
